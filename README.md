# Ciągła synchronizacja katalogów.
Program który dba o to, by zawartość katalogu u wszystkich klientów była taka sama. Zmiany plików są wykrywane natychmiastowo.
## Działanie
Po uruchomieniu i podaniu ścieżki do folderu który ma być synchronizowany klient może dodawać, modyfikować i usuwać z niego pliki. Serwer monitoruje te zmiany i wysyła je do innych połączonych klientów.
